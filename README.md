# Nodejs FES Template


# Pre-requisites
- Install [Node.js](https://nodejs.org/en/) version 16.3.0


# Getting started
- Clone the repository
```
git clone  <https://github.com/Guray00/GeoGraphix-backend.git> <GeoGraphix>
```
cd <GeoGraphix>
npm install
```
- Build and run the project
```
npm start -- npx tsc

npx tsc
```
  Navigate to `http://localhost:3001`
